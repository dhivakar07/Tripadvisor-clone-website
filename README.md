🌍 Tripadvisor Clone Website

📖 Overview
The Tripadvisor Clone Website is a front‑end project designed to replicate the look and feel of the popular travel platform. It demonstrates skills in UI/UX design, responsive layouts, semantic HTML, accessibility, and interactive JavaScript features.
This project is part of my portfolio to showcase professional coding practices and recruiter‑ready presentation.

✨ Key Features
🔷Homepage Layout inspired by Tripadvisor with hero banners, navigation, and search bar.
🔷Responsive Design optimized for desktop, tablet, and mobile screens.
🔷Interactive Search & Hover Effects built with JavaScript.
🔷Smooth Transitions & Animations for modern UI polish.
🔷Semantic HTML & Accessibility improvements for recruiter visibility.
🔷Reusable Components for scalable front‑end development.

🛠️ Tech Stack
🔷HTML5 → Semantic structure and accessibility
🔷CSS3 → Styling, Flexbox/Grid layouts, transitions, responsiveness
